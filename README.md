Carrossel com Botão
Projeto pessoal desenvolvido para praticar HTML, CSS e JavaScript.

##Funcionalidades

Avançar a imagem ao clicar em próximo
Voltar a imagem ao clicar em anterior

##Objetivo

Criar um carrossel de imagens funcional.

##Tecnologias

HTML
CSS
JavaScript
